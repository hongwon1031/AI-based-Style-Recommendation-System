# AI-체형보완 스타일 추천 시스템
본 프로젝트는 사용자의 체형을 분석하고, 이에 적합한 패션 스타일을 추천해 주는 웹 애플리케이션입니다.
딥러닝 기반의 인체 분할(Deeplabv3), 체형 분류(CNN) 모델을 사용하여 사용자 이미지를 분석하고,
HR-VTON(High-Resolution Virtual Try-On) 기법을 통해 가상 피팅을 제공합니다.
